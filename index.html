Langkah Penggunaan Sistem:

01 Pengaturan KOP Sesuaikan logo instansi, identitas daerah, nama dinas, email dan website resmi sekolah di tab Pengaturan KOP.
02 Daftar Mapel Kelola daftar Mata Pelajaran Wajib dan Muatan Lokal yang akan dimasukkan ke lampiran SKL.
03 Input Siswa & Nilai Masukkan data identitas siswa baru, lalu gunakan menu **Input Nilai** untuk melengkapi transkrip desimal siswa.
04 Cetak & Arsip Pilih siswa di tab Cetak SKL, cetak dokumen fisik, lalu kunci dan simpan seluruh database ke menu **Arsip Data** per tahun pelajaran.

Pengaturan KOP Surat Resmi

Sesuaikan kop surat yang dicetak secara resmi, termasuk logo kiri, kanan, and profil instansi.

Biarkan kosong untuk menggunakan watermark default.

Biarkan kosong jika tidak memerlukan logo kanan.

Pengaturan Format SKL

Data ini digunakan sebagai template header, nomor, penandatanganan dokumen, dan penanggalan.

Blok teks lalu klik tombol di atas untuk membuat teks tebal atau miring.

Daftar Data Siswa

Gunakan formulir untuk menambahkan data pribadi siswa beserta nilai kelulusannya.

Nama Lengkap Nomor Induk Siswa Nasional Tempat, Tgl Lahir Status Nilai Aksi

Belum ada data siswa terdaftar

Silakan klik "Tambah Data Siswa" untuk memulai.

Input Nilai Kelulusan Siswa

Pilih siswa terlebih dahulu, kemudian masukkan nilai desimal (dua angka belakang koma) untuk setiap mata pelajaran yang terdaftar.

Silakan daftarkan siswa terlebih dahulu di tab "Data Siswa".
